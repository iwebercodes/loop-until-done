# Color Palette

Derived from Ralph's CLI output. Dark mode aesthetic inspired by GitHub dark mode, Vercel, and Linear.

| Purpose | Hex Value | Notes |
|---------|-----------|-------|
| Background | #0D1117 | GitHub dark |
| Surface/cards | #161B22 | Slightly lighter than bg |
| Primary accent | #06B6D4 | Cyan |
| Success/CTA | #22C55E | Green |
| Highlight | #EAB308 | Yellow |
| Special | #D946EF | Magenta |
| Text primary | #E6EDF3 | Light text |
| Text secondary | #8B949E | Muted text |
| Border | #30363D | Subtle borders |
